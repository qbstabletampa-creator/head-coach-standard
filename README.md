# The Standard, Program Neutral Head Coach Bible

A clean, program-neutral HTML version of a head coach program bible.

## What is included

- Mission, vision, values
- The program standard
- Discipline philosophy
- Academics
- Strength and training staff
- Teaching philosophy
- Plan to win
- Scheme philosophy
- Staff hiring
- Recruiting philosophy
- Calendar and first steps
- Budget starter system
- Expansion roadmap for missing head coach systems

## Live site

Published through GitHub Pages.

## Notes

School-specific language has been removed. The core rules and philosophy are preserved in a neutral format.
